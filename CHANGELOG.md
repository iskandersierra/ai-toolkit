## [0.0.1] - 2026-05-24

### Added

* Added anchored annotations that you can create or edit directly from the editor and comment workflows.
* Added review session management, including active session switching plus clear and delete commands.
* Added annotation lifecycle actions for reanchoring, dismissing, resolving, and reopening tracked comments.
* Added draft output generation for the active review session in Markdown, JSON, and YAML.
* Added workspace-local annotation storage with schema validation and VS Code comment projection.
